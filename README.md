# MovieLibrary

Aplicación Android desarrollada en Kotlin usando arquitectura MVVM.

## Tecnologías usadas
- Kotlin
- Room Database
- LiveData
- ViewModel
- Navigation Component
- Safe Args
- RecyclerView

## Funcionalidades
- Ver lista de películas
- Agregar película
- Editar película
- Ver detalle
- Marcar como vista/no vista
- Persistencia local con Room
